World Wide Web

WWW is a collection of HTML documents

HTML stands for Hypertext markup language
Hyper links are used to connect all the documents.
Invented in 1990's and has 30 Billion pages.


Components of the Web

There are 4 Components of the web:
Browser
Internet
HTTP
Servers

Browser uses internet and requests for a HTML document. The internet uses HTTP(Hypertext transfer protocol) to request and fetch HTML documents from servers.
Servers are just computers but designed to store and transfer HTML documents not like our personal desktop to view the documents.

HTML(Hypertext markup language)

made up of
  • text content - what you see
  • markup - what it looks like
  • references to other documents - eg images and videos
  • links to other pages
