Components of the Web

There are 4 Components of the web:
Browser
Internet
HTTP
Servers

Browser uses internet and requests for a HTML document. The internet uses HTTP(Hypertext transfer protocol) to request and fetch HTML documents from servers.
Servers are just computers but designed to store and transfer HTML documents not like our personal desktop to view the documents.

HTML(Hypertext markup language)

made up of
  • text content - what you see
  • markup - what it looks like
  • references to other documents - eg images and videos
  • links to other pages
HTML Markup - Tags

An HTML element comprises of an opening tag and a closing tag and content in between the tags.

b stands for bold. em stands for italics.

HTML Attributes

anchor tag href attribute to link to other pages such as udacity.com or reddit.com.

Image tag has multiple attributes such as src which refers to url of the image and alt which is alternate text and is required for broken requests or blind.

HTML - CSS - DOM
HTML is a language and has its own syntax and rules. CSS is also an language to provide design to the HTML file. DOM is termed as Document Object Model which creates the tree structure for the html document.

Boxify the design : We should think of boxes and try to make as many boxes to make any design. Every thing in html file is a box even circle, ovals are inside a box. First we should make boxes around outer design and then make boxes around inner elements.
